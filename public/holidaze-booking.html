Overview

Holidaze is a booking platform for tourists and accodomation owners. The application was designed and developed for a real world client. Functional requirements was given, transfered into a Clickable UX Prototype that the client could test before development began. Then development began, and the project was done within 7 weeks.

Visit Project View Code

Technologies Used:

  • React
  • Tailwind CSS
  • SCSS
  • Node
  • Mongodb
  • Netlify

The application has a visitor side and customer side. The design was prototyped using Adobe XD then implemented within 7 weeks.
