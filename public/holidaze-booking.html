Overview

Holidaze is a booking platform for tourists and accommodation owners. The application was designed and developed for a real world client. Functional requirements was given, transferred into a Clickable UX Prototype that the client could test before development began.

Visit Project View Code

Technologies Used:

  • React
  • Tailwind CSS
  • SCSS
  • Node
  • Mongodb
  • Netlify

The application has a visitor side and customer side. The design was prototyped using Adobe XD then implemented within 7 weeks.
